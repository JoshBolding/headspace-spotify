# Asset Notice

Headspace Spotify is a fan/nostalgia project that rebuilds the experience of the classic Headspace media player skin as a modern Spotify desktop client.

This repository includes:

- The original Headspace skin archive under `assets/original`.
- Extracted original skin assets under `assets/original/extracted`.
- Converted runtime PNG assets under `assets/converted`.
- A small number of project-generated visual assets used for the face-alive Easter egg.

These assets are included to preserve and demonstrate the restoration/rebuild work. This project is not affiliated with Spotify, Microsoft, Windows Media Player, or the original Headspace skin authors.

If you own rights to any included asset and want it removed or credited differently, open an issue and it can be handled.
