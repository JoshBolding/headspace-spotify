# Discord Share Draft

I rebuilt the old Headspace alien-head media player skin as a tiny Spotify desktop client.

It keeps the weird compact alien-head chrome, drawers, speaker ears, and tiny transport controls from the original skin, but adds Spotify auth, playback controls, library browsing, queue, lyrics, themes, and audio-reactive visuals.

It is still rough around the edges, but the fun idea is: what if the old desktop skin era came back as modern music software?

Repo: https://github.com/JoshBolding/headspace-spotify

Setup note: you need your own Spotify app client ID and redirect URI. Local playback works best with Spotify Premium/Widevine; otherwise it falls back to Spotify Connect.
